# Términos de Servicio — Reino Oscuro Gaming

Página de Términos de Servicio para la comunidad gaming **Reino Oscuro**.

## Contenido

El documento cubre las siguientes secciones:

1. Aceptación de los Términos
2. Descripción del Servicio (foro, tienda y comentarios)
3. Registro y Cuenta de Usuario
4. Normas de Conducta
5. Foro de la Comunidad
6. Tienda y Pagos
7. Propiedad Intelectual
8. Limitación de Responsabilidad
9. Suspensión y Cancelación
10. Modificaciones
11. Contacto

## Estructura de archivos

```
/
├── terminos-de-servicio.html   ← Este archivo
├── politica-de-privacidad.html ← Documento relacionado
└── logo_rog.png                ← Logo requerido (debe estar en la misma carpeta)
```

## Requisitos

- El archivo `logo_rog.png` debe estar en la **misma carpeta** que el HTML.
- No se necesitan dependencias adicionales. Las fuentes se cargan desde Google Fonts (requiere conexión a internet).
- Compatible con todos los navegadores modernos.

## Personalización

| Qué cambiar | Dónde encontrarlo |
|---|---|
| Nombre de contacto o email | Sección XI — Contacto |
| Edad mínima de registro | Sección III — Registro |
| Política de devoluciones | Sección VI — Tienda y Pagos |
| Fecha de actualización | Cabecera del documento (`update-badge`) |
| Colores principales | Variables CSS en `<style>` (`:root`) |

## Enlace entre documentos

Este archivo enlaza automáticamente a `politica-de-privacidad.html` desde el footer. Ambos documentos deben estar en la misma carpeta para que el enlace funcione correctamente.
